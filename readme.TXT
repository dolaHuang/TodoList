概要：这是一个前端小练习，实现了简单的todolist功能
------------------------------
练习题名称：Todolist
------------------------------
练习所需知识点：HTML+CSS+JavaScript
------------------------------
练习实现的功能：1、在输入框中输入ToDo，单击“Add”添加ToDo到正在进行项目中
		2、单击多选框，将ToDo移动的已完成项目中
		3、单击ToDo后面的“X”可删除该条ToDo
		4、单击“全部清除”，删除所有ToDo
		5、正在进行和已经完成都提供实时计数功能，即实时显示当前条数
